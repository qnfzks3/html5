시멘틱 웹 관련 요소

개요

시멘틱 태그란 브라우저, 검색엔진 ,개발자 모두에게 콘텐츠의 의미를 명확히 설명하는 역할을 담당하는 요소

시맨틱 태그를 이용하면 컴퓨터가 HTML요소의 의미를 보다 명확히 해석하고 데이터로 활용할 수 있는 시맨틱 웹을 실현할 수 있음

시맨틱 태그 사용 이전 레이아웃 구성

시멘틱 태그 용어 설명

header : 웹 사이트 로고, 검색폼을 나타내는 영역

nav : 웹 사이트를 궝하는 메뉴gnb(global nevigation bar)를 나타내는 영역

main : 웹 사이트의 중요 컨텐츠를 표시하는 영역

section : 중요 컨텐츠를 구성하는 하위 컨텐츠를 그룹화 할 때 사용

article : 중요 컨텐츠들을 구성하는 하위 컨텐츠가 보여지는 영역

aside : 웹 사이트 중요 컨텐츠와 관련없는 내용을 표시하는 영역

footer : 웹 사이트 저작권이나 관리자 정보등을 나타내는 영역
